## Refine Do Task

Follow CLAUDE.md. Execute tasks from RefineProgress.md.

### Prerequisites

RefineProgress.md must exist. If not, run RefineWorkflow.md first.

### Task Selection

1. Respect priority order and dependencies
2. User may specify tasks directly
3. May group related tasks for batch execution

### Execution

1. Read task definitions
2. Ensure dependencies are completed
3. Execute Planning Phase for all selected tasks
4. Follow CLAUDE.md to complete the remaining phases(DO NOT SKIP UNLESS NOT REQUIRED)

### Escalation

If a task requires better planning, run RefineWorkflow.md with the improvement to update RefineProgress.md.
